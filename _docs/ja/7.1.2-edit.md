クラスターの基本情報、プロバイダ、モニタリング、クラスターのタイプを修正する機能である。 プロバイダのアカウント、タイプ、リージョンは変更できない。

### a\) クラスター → クラスター選択 → 登録 → 変更ボタン → 修正完了後、保存ボタンをクリック。
