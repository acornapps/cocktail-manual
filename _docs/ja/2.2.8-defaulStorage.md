기본 스토리지 지정 기능이다.
카탈로그 배포시 워크로드 타입이 StatefulSet 타입인 경우 수정없이 바로 배포할 경우 스토리지 볼륨이 생성되는 기본 위치이다.  
스토리지 플러그인 타입이 NFS Named 인 경우 기본 스토리지 설정은 불가능하다.

### a\) 클러스터 → 클러스터 선택 → 볼륨 → 볼륨 액션버튼 → 기본 스토리지 지정을 클릭한다.
